1. Install python 2.7.10 from the following link:
	https://www.python.org/downloads/

2. Install pip using the following two commands:
	curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
	python get-pip.py
	
3. Enable google drive api using the following link:
	https://developers.google.com/drive/api/v3/quickstart/python
	
4. Install google drive library by using the following command:
	pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
	
3. Install cryptography library using the following command:
	pip install cryptography
	
4. Install PBKDF2 library using the following command:
	pip install pbkdf2
	
5. Now run the python script named project.py and follow the instructions.
